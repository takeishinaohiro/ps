    reQuire::Schoolとは
    カリキュラムのないプログラミングスクール

    • やっぱり不安だよ…

      ご安心ください！reQuire::Schoolでは現役のプログラマーが一人一人に付き、どのような質問にも丁寧に解説します！reQuire::Schoolのメンターは全員スクールで学んだ経験があるため、皆さんの気持ちを理解して、丁寧に対応することができます！

    • 情報の使い方がわからない...

      そんな方のためにreQuire::Schoolの出番です！ reQuire::Schoolでは、現役のプログラマーが有益な情報を整理し、わかりやすくまとめることによって、誰でも理解できるようにしています！

    • カリキュラムがない？？

      現在世の中では、インターネットが大幅に普及し、たくさんの情報が溢れています。その中にはとてもわかりやすく有益なコンテンツや情報がたくさんあり、わからないこともすぐに解決できてしまいます。現役のプログラマーも日々ググって情報を調べながらプログラミングをしているのです！

    具体的に何を学ぶの？

    HTML

    HTMLとはWebページを作るための最も基本的なマークアップ言語です。
    HTMLを使うことで文字をWebページに反映させることができます。

    CSS

    CSSとはHTMLで書いた文字の色・大きさ・背景の色や配置などを指定し、
    装飾するための言語です。CSSを理解することで、おしゃれで見やすいWebサイトを作ることができます。

    jQuery

    jQueryを使うことで、webサイトのあらゆる動きや機能を取り付けることができます。
    jQueryを理解することで、利用する人が使いやすく楽しめるサイトを作成することが可能になります。

    このスキルがあると、一人でwebページが作れる！！
